## 📹 Script Prompt cho Video Nấu Ăn

### Cấu trúc Video Nấu Ăn (30-60 giây)

**Format Video:**
```
1. Opening (3-5 giây)
   - Hiển thị món ăn hoàn chỉnh
   - Tiêu đề của món ăn

2. Nguyên liệu (5-8 giây)
   - Quay các nguyên liệu được bố trí
   - Hiển thị từng thành phần

3. Quy trình nấu (15-40 giây)
   - Chuẩn bị
   - Nấu chính
   - Plating

4. Kết thúc (3-5 giây)
   - Hiển thị sản phẩm cuối cùng
   - GỌI TO ACTION (tính năng chưa có)
```

### Prompt Video cho Mỗi Món

#### **Phở Bò - Video Prompt:**
```
Create a short cooking video (45 seconds) for Vietnamese Pho Bo:
- 0-3s: Show finished bowl of pho with steam rising
- 3-8s: Display all ingredients arranged beautifully
- 8-15s: Show beef slices being added to hot broth
- 15-30s: Demonstrate noodles being added with herbs
- 30-40s: Show final plating and garnishing
- 40-45s: Close-up of the finished dish
Use warm, appetizing colors. Professional culinary video style.
```

## 🚀 Cách Sử Dụng từng Công Cụ

### **Sử dụng Midjourney:**

1. Join Midjourney Discord server
2. Đi tới channel `#newbies`
3. Gõ `/imagine prompt:[your_prompt]`
4. Chờ AI tạo ảnh (1-2 phút)
5. Upscale ảnh yêu thích
6. Download ảnh độ phân giải cao

```
/imagine prompt: A beautiful bowl of Vietnamese Pho Bo with aromatic broth...
```

### **Sử dụng DALL-E 3:**

1. Truy cập https://openai.com/dall-e-3
2. Login hoặc tạo tài khoản
3. Paste prompt vào ô text
4. Chọn style (Vivid hoặc Natural)
5. Click Generate
6. Download ảnh

### **Sử dụng Leonardo.ai:**

1. Truy cập https://leonardo.ai
2. Click "Create"
3. Chọn model "Photo Real" hoặc "Alchemy"
4. Paste prompt
5. Set quality: 750x750 hoặc cao hơn
6. Click Generate
7. Download ảnh

### **Sử dụng Runway ML:**

1. Truy cập https://runwayml.com/
2. Create new project
3. Upload ảnh công thức
4. Chọn "Motion" effect
5. Chỉnh duration: 10-15 giây
6. Generate video
7. Download

## 📝 Script Python để Cập Nhật Ảnh/Video vào Trang Web

Nếu bạn muốn tự động cập nhật, tạo file `generate_recipes.py`:

```python
import json

# Dữ liệu công thức với AI-generated images và videos
recipes_data = {
    "1": {
        "title": "Phở Bò",
        "image": "https://your-ai-generated-image-url.jpg",
        "video": "https://your-video-storage.com/pho-bo.mp4"
    },
    # ... thêm các công thức khác
}

# Cập nhật vào file recipes.json
with open('recipes.json', 'w') as f:
    json.dump(recipes_data, f)
```

## 💡 Mẹo & Tipscho Kết Quả Tốt

### Prompt Tips:
1. **Cụ thể hóa**: Càng chi tiết câu mô tả, kết quả càng tốt
2. **Thêm lighting**: "professional studio lighting", "natural daylight"
3. **Chất lượng**: Thêm "4k", "8k", "high resolution"
4. **Style**: Thêm "food photography", "cinematic", "appetizing"
5. **Composition**: Thêm "shot from above", "side view", "close-up"

### Best Practices:
- Tạo 3-4 biến thể mỗi ảnh, chọn cái tốt nhất
- Thử nhiều model khác nhau
- Sử dụng negative prompts: `avoid blurry, avoid low quality`
- Tạo style guide và áp dụng nhất quán

### Lưu ý Bản Quyền:
- Ảnh tạo bằng AI có thể sử dụng tự do (tuỳ tuân theo T&Cs của dịch vụ)
- Kiểm tra license của mỗi công cụ
- Ghi rõ nguồn ảnh trong trang web
